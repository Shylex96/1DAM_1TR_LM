Desglose de la página web sobre dónde están todas las cosas que se exigen en la práctica para puntuar según rúbrica.

1. Imágenes {
    "Algunas en tamaño original": Las puedes encontrar en -> adoptar.html, en medio del vídeo recortado de 12segundos y del audio. Debajo de la tabla y encima del footer. Es el perro con auriculares que conserva su tamaño original.
    "Algunas con tamaño especificado": Las puedes encontrar en -> En mascotas.html tenemos una galería de imágenes con 6 mascotas en la que las fotos son con una medida específica para adecuarlas al div. Alguna solo de width y otras con height tambien.
		"Con sus atributos": Las puedes encontrar en -> mascotas.html en la galería, en la que usé el atributo alt para definirles un nombre en caso de que no carguen, src para definir el source, y en el caso de estas mismas width y/o height para modificar su tamaño original.
}

2. Cabeceras {
    "Separar apartados con Cabeceras": Las puedes encontrar en -> index.html en el que vemos los títulos h1 (cabeceras) de "Nuestra historia" y "¿Por qué adoptar?"
}

3. Tablas {
    "Deben existir tablas": Las puedes encontrar en -> adoptar.html con las características de los animales.
}

4. Listas {
    "Deben existir listas": Las puedes encontrar en -> mascotas.html debajo de las fotos encontramos un pequeño párrafo descriptivo y luego una lista desordenada de 3 atributos sobre los animales.
}

5. Textos {
    "Deben existir diferentes tipos de textos": Las puedes encontrar en -> En general en toda la página, pero por indagar o concretar las tienes en index.html donde he usado negrita, cursiva, subrayado, he cambiado tamaños de fuente y tipografía.
}

6. Audio {
    "Debe haber un audio de al menos y no más de 5s": La puedes encontrar en -> index.html en la parte inferior de la página principal hay un botón (que me he permitido hacer el script de play y pause) que ejecuta un audio de ladrido. Tambien he agregado otro más clásico con los controladores normales en -> adoptar.html al lado de la foto del perro con auriculares. 
}

7. Video {
    "Debe haber un video": Lo puedes encontrar en -> adoptar.html contiene un vídeo descargado y recortado de 12segundos debajo del formulario.
}

8. Favicon {
    "Debe haber un Favicon": Lo puedes encontrar en las pestañas de la página en sí, es un icono de un perro y un gato en la misma imágen combinando la silueta blanca con la negra. Así mismo existe título de pestaña que pone "Adopta un animal". (Para mayor precisión consultar: index.html donde viendo el <head> podemos ver que está escrito y funciona).
}

9. Charset {
    "Debe contener Charset": Lo puedes encontrar en -> index.html en el apartado de <head> donde usé UTF-8.
}

10. Enlaces internos y externos {
	Enlace externo -> El vídeo de la página adoptar.html es un fragmento del vídeo original al que se puede acceder mediante el enlace de abajo que contiene el atributo _blank para llevar como enlace externo a otra página sin que el usuario pierda nuestra página.
	Enlace externo de nuestra misma página -> En cada una de las tres páginas tenemos en el nav tres enlaces (Inicio, Mascotas, Adoptar) que nos llevarán a estas web de nuestro propio sitio.
	Enlace Ancla (volver) -> En index.html podemos encontrar abajo un enlace interno llamado "Volver arriba ↗"  que nos lleva al inicio de la página. Además le agregué un atributo de smooth para que la subida no sea de golpe y quede brusco (aunque al ser tan pequeña la página, sigue quedando de golpe ya que no hay espacio suficiente para una subida suave).
}
